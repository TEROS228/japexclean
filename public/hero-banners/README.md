# Hero Banner Images

This folder contains banner images for the homepage hero slider.

## Image Requirements

- **Recommended Size**: 1920x800px (or similar 16:9 aspect ratio)
- **Format**: JPG or PNG
- **File Size**: Keep under 500KB for optimal loading speed

## Naming Convention

Use descriptive names:
- `banner-1.jpg`
- `banner-2.jpg`
- `banner-3.jpg`
- etc.

## Usage

Place your banner images here and they will be accessible at:
`/hero-banners/banner-1.jpg`

Example in code:
```jsx
<Image
  src="/hero-banners/banner-1.jpg"
  alt="Banner description"
  width={1920}
  height={800}
/>
```
